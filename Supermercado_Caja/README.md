## Descripción General del Sistema

Este sistema simula el funcionamiento básico de una **caja de supermercado**. Permite:

- Gestionar productos (Alta, Listado)
- Realizar ventas con carrito de compra
- Aplicar promociones y descuentos automáticos
- Generar tickets de venta
- Ver estadísticas básicas (productos más caros, stock bajo, etc.)
- Persistencia de datos en archivos

## Tecnologías utilizadas
- Python 3
- Archivos de texto para persistencia
- Modularización (separación en archivos)
